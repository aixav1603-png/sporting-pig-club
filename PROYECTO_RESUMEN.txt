   Archivo: public/styles.css (línea 6-14)
   
   :root {
     --primary-pig: #FF6B9D;        ← Color principal (rosa)
     --secondary-pig: #FFA6C1;      ← Color secundario (rosa claro)
     --accent-gold: #FFD700;        ← Acentos (oro)
   }

2. CAMBIAR TEMA COMPLETAMENTE
   Opción A: Usar TailwindCSS o Bootstrap
   Opción B: Crear nuevo styles.css
   Importante: Mantener los HTML id y clases que usa JavaScript

3. AGREGAR LOGO E IMÁGENES
   • Coloca archivos en: public/images/
   • Usa en HTML: <img src="images/logo.png" alt="Logo">

4. CAMBIAR TEXTOS
   Los textos están en los HTML. Edítalos directamente.

5. MANTENER LA FUNCIONALIDAD
   • Los id HTML son críticos para JavaScript
   • Si cambias un id, actualiza en: js/auth.js y js/dashboard.js
   • Las clases CSS pueden cambiar sin problemas

9. Usuario puede logout desde cualquier página
